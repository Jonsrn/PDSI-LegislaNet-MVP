# Fontes Inter - Instruções de Download

Para completar a configuração das fontes Inter, você precisa baixar os seguintes arquivos TTF:

## Arquivos Necessários:
- `Inter-Regular.ttf` (peso 400)
- `Inter-Medium.ttf` (peso 500)
- `Inter-SemiBold.ttf` (peso 600)
- `Inter-Bold.ttf` (peso 700)

## Como baixar:

1. **Opção 1 - Google Fonts (Recomendada):**
   - Acesse: https://fonts.google.com/specimen/Inter
   - Clique em "Download family"
   - Extraia o ZIP e copie os arquivos TTF necessários para esta pasta

2. **Opção 2 - Google Webfonts Helper:**
   - Acesse: https://gwfh.mranftl.com/fonts/inter?subsets=latin
   - Selecione os pesos: regular, 500, 600, 700
   - Baixe os arquivos TTF

3. **Opção 3 - Site Oficial:**
   - Acesse: https://rsms.me/inter/
   - Baixe a versão completa

## Após o download:
1. Coloque os 4 arquivos TTF nesta pasta (`fonts/`)
2. Execute `flutter pub get`
3. A fonte Inter estará disponível no app

## Nota:
O `pubspec.yaml` já está configurado para estas fontes.
